# 数组、链表、跳表

## 数组
时间复杂度
- prepend: O(1)，正常情况是O(1)的，可以在数组最开始预留一部分空间
- append: O(1)
- lookup: O(1)
- insert: O(n)
- delete: O(n)

## 链表
时间复杂度
- prepend: O(1)
- append: O(1)
- lookup: O(n)
- insert: O(1)
- delete: O(1)

## 跳表
### 原理及实现
**只能用于元素有序的情况**
可以看作是针对有序链表查询时间复杂度为O(n)的一种优化，即空间换时间

跳表（skip list）对标的是平衡树（AVL Tree）和二分查找，是一种 插入/删除/搜索 都是**O(log n)**的数据结构

最大的优势是原理简单、容易实现、方便扩展、效率更高

给有序链表加速，可添加多级索引

空间复杂度是O(n)

升维思想 + 空间换时间
